Bonekeep
Game prototype using a custom engine

Over the years I created many small protoype games as I was developing my custom OpenGL game engine. Bonekeep is one of such prototypes.

  • Very easy to use, minimalist, and cache-friendly Entity Component System
  • Resource manager for loading and unloading assets from memory
  • 2D animated sprite renderer
  • Support for audio playback / streaming
  • Collision system using AABB and spacial hashing for broad phase detection
