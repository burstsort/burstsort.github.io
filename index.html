Bonekeep
Game prototype using a custom engine | C++

Proof of concept game, made to test the capabilities of custom OpenGL game engine.

The engine features:

  • Minimalist and cache-friendly Entity Component System
  • Resource manager for efficiently loading and unloading assets from memory
  • Animated 2D sprite rendering
  • Support for audio playback / streaming
  • Collision system using AABB and spatial hashing for broad phase detection
